История изменений

Версия 1.0.6:

  • Добавлена поддержка сертификатов с атрибутами DN в формате определенном RFC 2253.

Версия 1.0.5:

  • Реализованы дополнительные бизнес проверки документов.
  • Добавлена поддержка cross-origin запросов
  • Изменена домашняя страница УТМ.
  • Изменены правила взаимодействия с ПО Terminal Updater.

Версия 1.0.4:

  • Исправлены проблемы зависания Procrun.
  • Изменен порядок логирования. Лог файлы сжимаются (zip) и хранятся не более трех дней.
  • Изменен функционал для работы с ПО Terminal Monitoring.
  • Добавлен функционал для работы с ПО Terminal Maintenance.
  • Изменена домашняя страница УТМ. Добавлена возможнось просмотра схем и сертификатов.
  • Добавлен функционал для обработки чеков пивной продукции.
  • Оптимизирована работа с памятью.

Версия 1.0.3:

  • Внесены изменения в инсталяционный файл для увеличения стабильности процесса установки.
  • Испрвлена ошибка выбора ключа при инициализации SSL контекста.
  • Добавлена возможность отключения отправки по HTTP.

Версия 1.0.2:

  • Добавлена возможность работы модуля через web-службу.
  • Документы отправляются по шифрованному каналу (SSL/TLS).
  • Документы пересылаются в подписаном (ЭЦП) конверте
  • Разворачивание и загрузка, в зависимости от разрядности платформы Windows, происходит автоматически.
  • Добавлена возможность настройки прокси соединения.
  • Добавлена обфускация.
  • Добавлен функционал по передаче/получению документов ЕГАИС опт.
  • Обновлена версия базы данных до 10.11.1.1.
  • Улучшены бизнес проверки тикета.
  • Реализован интерфейс доступа к документам посредством HTTP запросов
  • Исправлена ошибка запуска УТМ с несколькими подключенными хранилищами(токенами).
  • Добавлен вывод системной информации в лог.
  • В формат ответа терминала - внешнему источнику добавлена информация о произошедшей ошибке.
  • Добавлен функционал для работы с ПО Terminal Monitoring.
  • Добавлен функционал для работы с ПО Terminal Updater.
  • Усилена защита межпрограммного взаимодействия.
  • Решена проблема наличия нескольких ключей в хранилищах токена. Выбирается самый свежий.
  • Добавлена проверка CRL PKI сертификата.
  • Создан инсталяционный файл для платформы Windows.
  • Добавлениа возможность запуска УТМ как службы Windows.

Версия 1.0.1:

  • Добавлен offline режим отправки документов.
  • Улучшена система логирования событий.
  • Переход на XML формат документов.
  • Архитектура ПО преобрела модульную структуру.
  • Введена проверка возраста буфера (БД).

Версия 1.0.0:

  • Релиз первой версии универсального транспортного модуля.
  • ПО реализует передачу/получение документов ЕГАИС розница на сервер.
